# Feature Workflow

Work on ONE feature at a time. Do not start a new feature until the current one is complete.

## Definition of "complete":
- All tests pass
- Code is reviewed/approved
- Feature works as specified in PRD
- No known bugs

## When working on a feature:
- Focus only on that feature
- Don't refactor unrelated code
- Don't add "nice to have" improvements to other areas
- Complete it before moving on
